JPcap Library Directory
======================

Place the JPcap JAR file in this directory.

Download JPcap from:
- Official site: http://netresearch.ics.uci.edu/kfujii/jpcap/doc/
- Or search for "jpcap jar download"

Required file: jpcap.jar

After placing jpcap.jar in this directory, you can compile and run the project using the provided scripts.
